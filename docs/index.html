🔖 Bookmarklet refresher

For browsers without Tampermonkey (e.g. Opera). Install the bookmarklet once — it never needs updating. To sync all prices in one go, tap a Sync button below: it opens the store with your item list, then you tap the bookmarklet once and it grabs every price and saves them here. No token in the bookmark.

or drag this link to your bookmarks bar

Set up (once): Desktop — drag the link to the bookmarks bar. Android/Opera — bookmark any page, then edit it: name it PT Sync, paste the code as the URL.

Use: tap Sync Coles (or Woolworths) above → the store opens (solve the "I am human" check if shown) → tap your PT Sync bookmark once → it reads every tracked price and bounces back here to save them. New items are included automatically. Tip: if opened directly on a single product page, the bookmarklet just grabs that one item.
